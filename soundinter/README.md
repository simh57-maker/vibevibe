# Sound Interactive Media

마이크 입력의 데시벨에 반응하는 인터랙티브 미디어 웹사이트입니다.

## 주요 기능

- **마이크 입력**: 로컬 마이크를 통해 실시간 오디오 입력 수신
- **데시벨 반응형 명도**: 소리 크기에 따라 이미지/영상의 밝기가 자동으로 조절됩니다
- **미디어 업로드**: 이미지와 영상 파일을 업로드할 수 있습니다
- **비율 유지**: 업로드된 미디어는 원본 비율을 유지하며 화면에 꽉 차게 표시됩니다
- **민감도 조절**: 0-10 범위로 데시벨 반응 민감도를 조절할 수 있습니다
- **색상 필터**: multiply 블렌드 모드로 색상 필터를 적용할 수 있습니다
- **UI 토글**: `/` 키를 눌러 컨트롤 패널을 숨기거나 표시할 수 있습니다

## 로컬 서버 실행 (권장)

마이크 권한을 위해 로컬 서버를 통해 실행하는 것을 권장합니다.

### Node.js가 설치되어 있는 경우:

```bash
# 서버 시작
node server.js

# 또는
npm start
```

그 다음 브라우저에서 `http://localhost:8000` 접속

### Python이 설치되어 있는 경우:

```bash
# Python 3
python3 -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

그 다음 브라우저에서 `http://localhost:8000` 접속

## 사용 방법

1. 위의 방법으로 로컬 서버를 실행하고 브라우저에서 접속합니다
2. "미디어 업로드"에서 이미지 또는 영상 파일을 선택합니다
3. "마이크 시작" 버튼을 클릭하여 마이크 권한을 허용합니다
4. 소리를 내면 업로드한 미디어의 밝기가 반응합니다
5. 민감도를 조절하여 반응성을 변경할 수 있습니다
6. 색상 필터를 추가하여 시각적 효과를 더할 수 있습니다
7. `/` 키를 눌러 UI를 숨기고 전체 화면으로 감상할 수 있습니다

## 디버깅 정보

마이크 설정 섹션에서 실시간으로 다음 정보를 확인할 수 있습니다:
- **RMS**: 평균 음량 (Root Mean Square)
- **Peak**: 순간 최고 음량
- **Freq**: 주파수 영역 분석 음량

소리가 감지되지 않는다면:
1. 브라우저 개발자 도구(F12) → 콘솔 탭에서 "마이크 시작됨" 메시지 확인
2. RMS, Peak, Freq 값이 변하는지 확인
3. 민감도를 8-10으로 높여보세요

## 민감도 설정

- **0**: 가장 둔감 (큰 소리에만 반응)
- **5**: 기본값
- **10**: 가장 민감 (작은 소리에도 쉽게 반응)

## 기술 스택

- MediaStream API

## 브라우저 호환성

Chrome, Firefox, Safari, Edge 등 최신 브라우저에서 사용 가능합니다.
마이크 권한이 필요하므로 HTTPS 또는 localhost 환경에서 실행해야 합니다.
