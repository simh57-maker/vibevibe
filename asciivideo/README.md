# ASCII Video Converter

영상을 아스키 그래픽으로 변환하는 웹 애플리케이션입니다.

## 기능

### 1. 영상 업로드
- 비디오 파일을 업로드하면 자동으로 재생되며 실시간으로 아스키 그래픽이 적용됩니다.

### 2. 아스키 그래픽 효과
- 영상 프레임을 흑백으로 처리 후 명암에 따라 4단계로 분류
- 각 단계별 기본 문자: !, @, #, $
- 각 단계마다 원하는 텍스트로 변경 가능 (최대 5글자)
- 체크박스로 각 단계를 보이기/숨기기 가능
- 기본 서체: Roboto Mono

### 3. 컬러 조정
- **텍스트 컬러**: HSB 값으로 조정 (Hue, Saturation, Brightness)
- **배경 컬러**: HSB 값으로 조정
- 실시간 컬러 프리뷰 제공

### 4. 영상 출력
- 효과가 적용된 영상을 WebM 형식으로 출력
- 원본 영상의 크기와 사이즈 유지
- 30fps로 녹화

## 사용 방법

1. 브라우저에서 `index.html` 파일을 엽니다.
2. "Upload Video" 버튼을 클릭하여 비디오 파일을 업로드합니다.
3. 업로드된 영상이 자동으로 재생되며 아스키 효과가 적용됩니다.
4. 우측 패널에서 설정을 조정합니다:
   - 각 명암 단계의 문자 변경
   - 체크박스로 특정 단계 숨기기/보이기
   - HSB 슬라이더로 텍스트 및 배경 색상 조정
5. "Export to MP4" 버튼을 클릭하여 결과물을 다운로드합니다.

## 기술 스택

- HTML5
- CSS3
- JavaScript (Vanilla)
- Canvas API
- MediaRecorder API

## 브라우저 호환성

- Chrome/Edge: 완전 지원
- Firefox: 완전 지원
- Safari: MediaRecorder API 지원 제한적

## 참고사항

- 영상 출력은 WebM 형식으로 제공됩니다 (브라우저 제약사항)
- 긴 영상의 경우 출력에 시간이 걸릴 수 있습니다
- 최상의 결과를 위해 고해상도 영상 사용을 권장합니다
